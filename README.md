## 📖 About

Artisto 2026 is a standalone creative companion tool designed to streamline your digital art workflow. Whether you're sketching, painting, or editing, this utility offers a suite of enhancements to boost productivity and unlock new creative possibilities. Built for Windows and MacOS, it provides a clean, non-intrusive experience without modifying any system files.

- **Operating System:** Windows 10 or later (64-bit) / MacOS 11 Big Sur or later
- **Processor:** Intel Core i3 / AMD Ryzen 3 or equivalent (Apple Silicon supported)
- **Memory:** 4 GB RAM minimum (8 GB recommended)
- **Storage:** 200 MB  disk space
- **Internet:** Required for initial  and optional updates
